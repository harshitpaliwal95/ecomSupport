# Ecom backend

### Tech

- Python3
- Django Framwork
- Sql Db ( PostgrasSQL v - 15.1 )
- Server AWS

## Get Started

- Clone

```
https://github.com/harshitpaliwal95/ecomSupport.git
```

- Go to the project directory

```bash
  cd ecomSupport
```

- Start the server

```bash
  python manage.py runserver
```
